# Excel_project_3
**Getting  shipdate with order id using xlookup**

>=XLOOKUP(D4,Table1[Order ID],Table1[Ship Date])

**Sales quantity of a particular category by month wise**

>=COUNTIFS(Table1[Order_month],D7,Table1[Category],E7)

**Total Revenue generated  of a particular category by month wise**

>=SUMIFS(Table1[Sales],Table1[Order_month],D9,Table1[Category],E9)

**Sales quantity of a particular category by region wise**

>=COUNTIFS(Table1[Region],D11,Table1[Category],E11)

**Total Revenue generated of a particular category by region wise**

>=SUMIFS(Table1[Sales],Table1[Region],D13,Table1[Category],E13)

**Average orders per customer**

>=AVERAGEIF(Table1[Customer Name],D15,Table1[Sales])

**which country does customer belongs to using index and match function**

>=INDEX(Table1[City],MATCH(D17,Table1[Customer Name],0))

**Sales quantity by region wise**

>=COUNTIF(Table1[Region],I4)

**Total revenue generated by region**

>=SUMIF(Table1[Region],I6,Table1[Sales])

**Checking mode of transaction customer used**

>=XLOOKUP(I8,Table1[Customer Name],Table1[Ship Mode],0)

**How many orders placed  in the particular date**

>=COUNTIF(Table1[Order Date],I10)

**Total revenue generated on that date**

>=SUMIF(Table1[Order Date],I12,Table1[Sales])

**Checking Sales quantity by Ship mode**

>=COUNTIF(Table1[Ship Mode],I14)
